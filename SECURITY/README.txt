This folder contains a home-made security system project.

Required Hardware:
- RaspberryPi
- Logitech 270 or similar camera
- Infrared Sensor (https://www.digikey.com/en/products/detail/osepp-electronics-ltd/PIR-01/11198651) or similar
- WiFi for Pi to report with

Once running, the security system will take a picture if motion is sensed and attempt to send an alert via text, email and tweet.

Basic features implemented - Twitter portion left out after struggle with approval process for known botting.

Consider tweaking alert style or routine to make it your own! Enjoy!
